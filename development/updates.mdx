<Update label="2025-03-26" description="Kotlin SDK 0.4.0 リリース">

- Claude Desktopとの統合とサンプルを追加
- 完全な変更履歴はこちらで確認できます: https://github.com/modelcontextprotocol/kotlin-sdk/releases/tag/0.4.0
</Update>

<Update label="2025-02-14" description="Java SDK リリース">
  - VMware TanzuのSpring AIによって開発されたJava SDKが、MCPの公式
    [Java SDK](https://github.com/modelcontextprotocol/java-sdk)としてリリースされました。
    これは既存のKotlin SDKに加えてサポート言語が増えたものです。
    Spring AIチームがSDKをメンテナンスし、Model Context Protocol組織の重要な一部として管理します。
    MCPコミュニティへの参加を歓迎します！
</Update>
